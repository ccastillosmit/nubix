# NEXO Consultoría

Sitio web institucional de NEXO Consultoría.

Consultoría interdisciplinaria en ingeniería, economía y datos, enfocada en ayudar a empresas a mejorar sus procesos, costos y toma de decisiones mediante soluciones técnicas y analíticas.

## Servicios
- Automatización de procesos y datos
- Reporting y Business Intelligence
- Modelos de costos y planificación
- Ingeniería aplicada a la gestión
- Gobierno de datos

## Tecnologías
- HTML5
- CSS3
- GitHub Pages
